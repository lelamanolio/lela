# I made my own VS Code theme (and you can try it)

I've been using <a href="https://marketplace.visualstudio.com/items?itemName=sdras.night-owl" target="_blank" rel="noopener noreferrer">Night Owl</a> for years. Great theme — but after a while, all that blue started to feel a bit cold.  
I wanted something darker, warmer, and honestly just... greener.

The problem?  
Most dark themes are just black with colored syntax on top. Nothing wrong with that, but I couldn't find anything that felt like sitting in a forest at midnight. So I built it myself.

![Preview JS](/projects/js.png)

**Midnight Grove** is a forest-inspired dark theme built around deep greens and soft accent colors.  
Calm enough for long sessions, readable enough to actually see what you're doing.

It's on the VS Code Marketplace — <a href="https://marketplace.visualstudio.com/items?itemName=lela.midnight-grove" target="_blank" rel="noopener noreferrer">grab it here</a>.

Enjoy coding in the grove. 🌲
